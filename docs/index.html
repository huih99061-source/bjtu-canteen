核心功能

覆盖食堂就餐全流程的可视化仿真

🪑

实时座位平面图

80 桌 × 4 座共 320 个座位，点阵视图与动态热力图两种模式实时展示占用状态。

📋

窗口排队可视化

7 个窗口队列人数实时更新，颜色区分拥挤程度，附带历史趋势折线图。

🕐

多速仿真时钟

支持 0.5× 至 60× 调速，完整模拟早餐、午餐、晚餐三个高峰时段的人流规律。

🍽️

菜品菜单与评价

点击窗口查看完整菜单与价格，提交评分和文字评价，数据持久化存储。

🌐

离线本地仿真

后端不可用时自动切换前端仿真模式，无需网络也可完整演示系统功能。

⌨️

快捷键操作

空格键暂停/继续，R 键重置，方向键调速，支持点击空座手动占座。
